# Data-Quality-Assessment-Preprocessing

This project focuses on analyzing and improving data quality using the Titanic dataset. It demonstrates essential preprocessing steps required before applying machine learning techniques.

## Tasks Covered

* Identifying data quality issues (missing values, duplicates, inconsistencies)
* Handling missing values using appropriate strategies
* Detecting and removing outliers using the IQR method
* Normalizing numerical features using Min-Max and Z-score scaling
* Applying PCA for dimensionality reduction based on feature correlation

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib & Seaborn
* Scikit-learn

## Dataset

* Titanic Dataset (Kaggle)

## Key Outcome

The dataset was cleaned, transformed, and prepared for further analysis. PCA was successfully applied to reduce dimensionality while preserving important information.
